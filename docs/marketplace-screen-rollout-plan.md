# Marketplace Screen Rollout Plan

## 1) User Journey Screen Map

| Journey          | Primary Screens (Current)                                                                                | Primary Goal                                                                      |
| ---------------- | -------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| Onboarding       | `LoginScreen` → `RegisterScreen` → `WalletSetupScreen`                                                   | Create session, create profile, optionally set wallet before entering marketplace |
| Discovery        | `HomeScreen` (`/feed`), `SearchScreen` (`/search`), `MapScreen` (exists, not yet routed in main shell)   | Find relevant actions quickly by proximity, category, and intent                  |
| Action detail    | `ActionDetailScreen`                                                                                     | Evaluate an action: criteria, location, reward, creator credibility               |
| Verification     | `VerificationCaptureScreen` → `VideoRecordingScreen` → `VideoReviewScreen` → `SubmissionStatusScreen`    | Capture evidence, submit proof, receive verification outcome                      |
| Rewards / wallet | `WalletSetupScreen`, `WalletScreen`, `RewardsScreen`, `CreateRewardPoolScreen`, `RewardPoolDetailScreen` | Manage balances, pools, and reward history                                        |
| Profile / social | `ProfileScreen`, `EditProfileScreen`, `UserProfileScreen`                                                | Show identity, progress, earned rewards, social graph                             |
| Creator flows    | `CreateActionScreen` (+ reward-pool screens above)                                                       | Publish actionable tasks, define verification, fund rewards                       |

## 4) Immediate High-Impact Screens To Integrate Next

1. `HomeScreen` and `SearchScreen` with real `action`/`category` data (highest marketplace discovery impact).
2. `ActionDetailScreen` with real action, steps, reward pool, and creator data.
3. `VerificationCaptureScreen` + `VideoReviewScreen` + `SubmissionStatusScreen` wired to `submission`/`verification`/`attestation` endpoints.
4. `CreateActionScreen` (including optional AI assist via `ai_action_endpoint.dart`) to unlock supply side.
5. `WalletSetupScreen` and `WalletScreen` for reward payout readiness.
6. `CreateRewardPoolScreen` and `RewardPoolDetailScreen` for creator-funded incentives.
7. `ProfileScreen`, `EditProfileScreen`, and `UserProfileScreen` for retention and social trust.
8. `MapScreen` route integration into discovery once live location/action joins are enabled.
